Que faire si le pocket WiFi ne fonctionne pas

Fiche simple pour vérifier rapidement le boîtier WiFi, identifier le problème principal et trouver une solution de secours sans perdre de temps.

Résumé rapide

Avant de paniquer, vérifier dans cet ordre : batterie, allumage, voyants, connexion de l'appareil, mot de passe, puis documents de commande. En parallèle, garder l'eSIM Ubigi comme solution de secours si besoin.

Vérifications immédiates

  1. Vérifier que le pocket WiFi est bien allumé et suffisamment chargé.
  2. Regarder si un voyant réseau ou batterie indique un problème évident.
  3. Essayer de couper puis rallumer le boîtier.
  4. Sur le téléphone, oublier le réseau WiFi puis se reconnecter avec le bon mot de passe.
  5. Tester avec un autre téléphone pour savoir si le souci vient du boîtier ou d'un appareil précis.

Plan B rapide

  • Si le pocket WiFi ne fonctionne pas immédiatement, basculer sur l'eSIM Ubigi si elle a été préparée.
  • Garder accessibles les mails de confirmation et de reçu pour toute demande d'assistance.
  • Faire les premières démarches essentielles avec la solution la plus rapide, puis traiter le problème plus calmement une fois installé.

Documents utiles

  • Mail de confirmation pocket WiFi
  • Mail eSIM
  • Reçu Ubigi / eSIM
