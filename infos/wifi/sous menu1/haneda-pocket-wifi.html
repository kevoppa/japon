Comment obtenir le WiFi à l'arrivée à Haneda ?

L'aéroport de Haneda propose un WiFi gratuit dans ses terminaux. Voici les étapes utiles, les bons repères visuels et les plans pour se connecter rapidement dès l'arrivée.

Résumé rapide

Le réseau gratuit de l'aéroport s'appelle HANEDA-FREE-WIFI. À l'arrivée internationale, tu passes par le Terminal 3, puis tu arrives dans le hall des arrivées 2F après la douane. Il suffit d'activer le WiFi, de sélectionner ce réseau, d'ouvrir un navigateur et de suivre les instructions à l'écran.

Images repères utiles

Étapes pour se connecter

  1. Après l'atterrissage, terminer le parcours normal : immigration, récupération des bagages, puis douane.
  2. Une fois arrivé dans le hall des arrivées du Terminal 3, sortir le téléphone.
  3. Activer le WiFi dans les réglages de l'appareil.
  4. Dans la liste des réseaux disponibles, sélectionner HANEDA-FREE-WIFI.
  5. Ouvrir un navigateur internet si la page de connexion ne s'affiche pas toute seule.
  6. Suivre les indications affichées à l'écran pour valider la connexion.
  7. Une fois connecté, utiliser Internet pour vérifier l'itinéraire, les mails, les billets ou prévenir les autres.

Où tu seras exactement

  • À l'arrivée internationale, vous passez par le Terminal 3 de Haneda.
  • Le WiFi gratuit est disponible dans le terminal.
  • Le repère principal après la douane est le 2F Arrival Lobby, c'est-à-dire le hall des arrivées au 2e étage.
  • Le plan d'arrivée permet de visualiser le trajet jusqu'au hall, puis le plan 2F permet de mieux se repérer une fois sorti.

Points importants

  • Le nom exact du réseau à choisir est HANEDA-FREE-WIFI.
  • Si rien ne s'affiche après la connexion, ouvrir manuellement Safari, Chrome ou un autre navigateur.
  • Le signal peut être plus faible dans certaines zones.
  • Le site officiel précise aussi que ce WiFi ne peut pas être utilisé dans les boutiques, sauf dans les lounges.
  • Le WiFi de l'aéroport est utile pour un besoin rapide à l'arrivée, mais il peut être moins stable qu'une eSIM ou un pocket WiFi personnel.
