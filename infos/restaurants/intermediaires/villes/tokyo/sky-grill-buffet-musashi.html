Sky Grill Buffet Musashi (Asakusa View Hotel 26F)

Grand buffet avec vue sur Tokyo Skytree, situé au 26e étage de l'Asakusa View Hotel, pratique pour un groupe avec réservation en ligne.

Galerie photos

Résumé

Sky Grill Buffet Musashi est une adresse efficace pour un groupe de 5 à Tokyo : buffet, grande vue en hauteur, service continu lunch/dinner et réservation web directe.

Infos clés

  • Nom : Sky Grill Buffet Musashi / Asakusa View Hotel
  • Adresse : 3-17-1 Nishiasakusa, Asakusa View Hotel 26F, Taito-ku, Tokyo, Japon
  • Téléphone : 03-3842-3376 (ligne directe TableCheck) ; réservations officielles 03-3842-6341 / 03-3842-2124 / 03-6736-6267 (10:00-17:00)
  • Horaires observés (site officiel) : petit-déjeuner 07:00-09:30, lunch 11:30-15:00, dinner 17:00-21:00 (semaine) / 17:00-21:30 (sam-dim-fériés)
  • Créneaux observés (TableCheck) : lunch semaine dernier accueil 13:30 ; lunch week-end/férié 11:30-13:00 ou 13:30-15:00 ; dinner semaine dernier accueil 19:00 ; dinner week-end/férié 17:00-19:00 ou 19:30-21:30
  • Capacité : non communiquée sur les pages officielles consultées
  • Fumeur : non-fumeur (indication TableCheck)
  • Réservation : recommandée ; réservation web disponible (jusqu'à 16 personnes sur formulaire public TableCheck)

Pourquoi on le met au programme

  • Vue Tokyo Skytree et skyline, utile pour une soirée repas + panorama.
  • Format buffet pratique pour un groupe avec goûts différents.
  • Réservation web simple depuis la page officielle.
  • Localisation claire à Asakusa View Hotel (26F), facile à noter dans le planning.

Prix (JPY / EUR)

  • Budget lunch observé (TableCheck) : 7,000 JPY ≈ 38.25 EUR
  • Budget dinner observé (TableCheck) : 7,000 JPY ≈ 38.25 EUR
  • Repère groupe 5 personnes (sur budget observé) : 35,000 JPY ≈ 191.24 EUR

Conversion indicative sur base 1 EUR = 183.02 JPY (BCE du 04/03/2026). Taux variable.

Conseils famille (5 personnes)

  1. Réserver en avance, surtout en dinner week-end/férié avec créneaux fixes.
  2. Prévoir un horaire d'arrivée en avance (annulation possible si retard important non signalé).
  3. Si vous voulez la vue, préciser la demande lors de la réservation (certaines places n'ont pas de panorama).
  4. Conserver l'adresse + lien Maps sur chaque téléphone avant le départ.
  5. Vérifier les menus/conditions du jour (saisonnalité et disponibilité produits).
