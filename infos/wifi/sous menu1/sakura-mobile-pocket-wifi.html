Comment récupérer la Pocket WiFi Sakura Mobile à l’arrivée à Haneda ?

Guide pratique complet pour récupérer facilement la Pocket WiFi Sakura Mobile à l’aéroport de Haneda, l’activer en quelques minutes, puis la retourner correctement le jour du départ pour la France.

Voir la page officielle Sakura Mobile Voir le plan officiel du Terminal 3 - Arrivées 2F Voir le plan officiel du Terminal 3 - Départs 3F

Résumé ultra rapide

À Haneda, le retrait de la Pocket WiFi Sakura Mobile se fait au comptoir JAL ABC, dans le Terminal 3, Arrival Lobby (hall des arrivées), 2F. Après l’immigration, les bagages et la douane, il faut simplement sortir dans le hall des arrivées puis rejoindre ce comptoir pour présenter la réservation.

Important : le comptoir de retrait est fermé entre 1h00 et 4h00 du matin. Si votre avion atterrit très tard, gardez bien ce créneau en tête.

1. Avant de partir pour le Japon

Ce qu’il faut préparer

  • Le mail de confirmation Sakura Mobile
  • Le numéro de commande
  • Le nom utilisé pour la réservation
  • Ton passeport

Le plus simple est d’avoir le mail de confirmation déjà ouvert sur ton téléphone, ou enregistré en PDF pour y accéder sans chercher dans ta boîte mail à l’arrivée.

Conseil utile

Juste avant le départ, fais une capture d’écran du mail de confirmation. Comme ça, même si ton téléphone met un peu de temps à retrouver une connexion, tu auras quand même toutes les infos sous la main.

2. Les étapes à suivre après l’atterrissage

Étape 1 — Descendre de l’avion

À l’arrivée internationale à Haneda, tu suis les panneaux pour l’entrée sur le territoire.

Étape 2 — Passer l’immigration

Dirige-toi vers Immigration / Passport Control avec ton passeport et les documents demandés.

Étape 3 — Récupérer les bagages

Va au tapis indiqué pour ton vol dans la zone Baggage Claim.

Étape 4 — Passer la douane

Une fois la douane passée, tu entres dans le hall des arrivées 2F. C’est seulement à partir de là que tu peux rejoindre le comptoir JAL ABC.

Étape 5 — Aller au comptoir JAL ABC

Cherche le JAL ABC Counter dans le hall des arrivées du Terminal 3. C’est le point officiel indiqué par Sakura Mobile.

3. Comment récupérer la Pocket WiFi au comptoir

  1. Approcher du comptoir JAL ABC.
  2. Dire que tu as une réservation Sakura Mobile.
  3. Montrer ton passeport.
  4. Montrer le mail de confirmation ou le numéro de commande.
  5. Récupérer le kit complet.

Phrase simple à dire au comptoir :
"Hello, I have a Sakura Mobile Pocket WiFi reservation."

Vérifier tout de suite le contenu du kit

  • Le routeur Pocket WiFi
  • Le câble de charge
  • La batterie ou l’alimentation si fournie
  • L’enveloppe de retour prépayée
  • Les identifiants WiFi si besoin

N’attends pas d’être dans le train pour vérifier le matériel. Fais-le directement au comptoir pour pouvoir demander de l’aide immédiatement si quelque chose manque.

4. Comment activer la Pocket WiFi

Dans la plupart des cas, la Pocket WiFi Sakura Mobile est déjà prête à l’emploi. Il suffit surtout de l’allumer et de connecter les téléphones dessus.

  1. Maintenir le bouton Power quelques secondes pour l’allumer.
  2. Attendre environ 30 secondes à 1 minute.
  3. Sur le téléphone, ouvrir les réglages WiFi.
  4. Repérer le nom du réseau WiFi affiché sur l’appareil ou sur l’étiquette.
  5. Entrer le mot de passe indiqué.
  6. Ouvrir Google Maps ou Safari/Chrome pour vérifier que tout fonctionne.

Exemple d’infos à retrouver sur le boîtier

SSID : SakuraMobile_xxxx

Password : xxxxxxxx

Bon réflexe juste après l’activation

Connecte immédiatement au moins deux téléphones du groupe. Comme ça, vous savez tout de suite si la connexion est stable avant de quitter l’aéroport.

5. Conseils d’utilisation pendant le voyage

  • Recharger la Pocket WiFi tous les soirs.
  • La garder dans un sac facile d’accès pendant les trajets.
  • Éviter qu’une seule personne parte avec si tout le groupe en dépend.
  • Faire une capture du SSID et du mot de passe pour les autres voyageurs.
  • Vérifier la batterie avant une grosse journée de visite.

Organisation conseillée pour votre groupe

Donnez la Pocket WiFi à la personne la plus souvent au centre du groupe, ou prévoyez une petite batterie externe dédiée. C’est ce qui évite le plus de galères pendant les journées longues.

6. Le jour du retour en France

Le retour ne se fait pas au comptoir de retrait. Pour Sakura Mobile, il faut remettre le matériel dans l’enveloppe de retour prépayée, puis déposer cette enveloppe dans une boîte postale Japan Post.

Ne compte pas sur le comptoir de l’aéroport pour reprendre l’appareil : Sakura précise que les comptoirs d’aéroport sont prévus pour le pickup, pas pour accepter les retours.

Ce qu’il faut remettre dans l’enveloppe

  • Le routeur Pocket WiFi
  • Le câble
  • Les accessoires fournis
  • La batterie si elle était incluse séparément

7. Comment retourner correctement la Pocket WiFi

  1. Éteindre la Pocket WiFi avant de partir pour l’aéroport.
  2. Rassembler tout le matériel du kit.
  3. Mettre l’ensemble dans l’enveloppe de retour prépayée.
  4. Bien fermer l’enveloppe.
  5. Déposer l’enveloppe dans une boîte postale Japan Post.

Sakura Mobile indique que l’enveloppe doit être postée avant minuit le jour de retour.

Où poster l’enveloppe à Haneda

Le plus prudent est de chercher une boîte postale dans les zones publiques du Terminal 3 avant de passer la sécurité.

Repère important

Les plans officiels Haneda affichent la présence d’une Post Box dans les services du terminal. Vérifie sur le plan 2F ou 3F selon ta circulation dans l’aéroport.

9. FAQ express

On récupère la Pocket WiFi avant ou après la douane ?

Après la douane, dans le hall des arrivées du Terminal 3.

Est-ce qu’il faut l’activer au comptoir ?

En général non. Il suffit surtout de l’allumer et de connecter les téléphones dessus.

Peut-on la rendre directement au comptoir le jour du départ ?

Non, le retour se fait via l’enveloppe prépayée déposée dans une boîte postale.

Quel est le meilleur moment pour poster l’enveloppe ?

Le plus sûr est de le faire à l’aéroport avant le passage de la sécurité.
